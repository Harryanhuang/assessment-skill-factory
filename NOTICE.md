# Notice

This repository contains an original workflow framework for building assessment skills from official source documents.

It does not include official PDFs, official exam papers, mark schemes, copyrighted rendered pages, or full official syllabus text.

References to AP, CAIE, Edexcel, AQA, IB, HKDSE and other education systems are used only to describe supported source-document profiles and pilot workflow examples. All trademarks, official documents and copyrighted materials remain the property of their respective owners.

Pilot artifacts are intended as structured examples of the workflow. They should be checked against the relevant official source before production use.
