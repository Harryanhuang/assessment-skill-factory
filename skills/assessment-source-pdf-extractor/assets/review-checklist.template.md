# Source Review Checklist

## PDF Identity

- [ ] title matches expected official document
- [ ] document_version recorded and matches PDF
- [ ] file_checksum_sha256 recorded or reason documented
- [ ] total_pages recorded and matches PDF
- [ ] local_archive_ref or official_url recorded
- [ ] PDF permission status recorded (encrypted/copy/print)
- [ ] local filename years checked against official title/version/valid years
- [ ] identity conflicts recorded in Risks

## Section Map

- [ ] contents / section map covers all relevant chapters
- [ ] admin / marketing / support / copyright sections marked as not extracted
- [ ] high-impact sections identified

## High-Impact Sections

- [ ] assessment overview / subject content table
- [ ] paper rules / tier rules
- [ ] bold/P reference / paper split / availability
- [ ] required practical / command words
- [ ] AO/skills / grade descriptions
- [ ] out-of-scope / forbidden combinations / issue/change summary

## Page-Map Traceability

- [ ] each high-impact target maps to correct page/section
- [ ] topic/content mappings sampled (10%-20% minimum)
- [ ] high-risk pages (tables, two-column, scans, OCR, formulae) fully reviewed
- [ ] rendered PNG samples saved for high-impact pages
- [ ] printed page refs cross-checked against physical page rendered files
- [ ] printed/physical page offsets recorded when present

## Layout Semantics

- [ ] tables do not lose cross-page meaning
- [ ] two-column layout preserves left/right distinction
- [ ] bold / P reference / footnotes preserved
- [ ] required practical / command words / formulae lists captured

## Confidence & Verdict

- [ ] extraction_confidence justified
- [ ] blocking_risks empty or documented
- [ ] verdict: pass / pass_with_risks / fail
